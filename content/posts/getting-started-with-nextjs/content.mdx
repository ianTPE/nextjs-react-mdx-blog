# 開始使用 Next.js 15

Next.js 15 正式登場，帶來更強大、更高效、更現代化的 React 全端開發體驗！本篇將帶你快速了解 Next.js 15 的最新特性、安裝方式與最佳實踐，並詳細解析 15 版相較於過去的重大優勢。

## Next.js 15 的重大優勢與新特性

Next.js 15 在原有 App Router、Server Components、Edge Rendering 等基礎上，進一步強化了開發效率、效能與 DX（Developer Experience）。主要亮點如下：

- **穩定的 Partial Prerendering（PPR）**：支援頁面局部預渲染，讓動態內容與靜態內容完美結合，極速首屏、即時互動。
- **更快的 Turbopack**：Turbopack 編譯器正式進入穩定版，開發啟動與熱更新速度大幅提升。
- **Edge Runtime 強化**：API 路由、Middleware、Server Actions 全面支援 Edge Runtime，延遲更低、可擴展性更強。
- **Server Actions 正式版**：支援直接在伺服器端執行 mutation/action，簡化表單、資料提交與安全性管理。
- **MDX 2.0 支援**：原生支援 MDX 2.0，React 組件與 Markdown 深度融合。
- **更簡潔的 Metadata API**：SEO 與 Open Graph 設定更簡單。
- **升級 React 19 支援**：自帶最新 React 19，享受更佳 Concurrent Features。
- **開發體驗提升**：錯誤提示、型別推斷、文件生成等 DX 全面升級。

## 什麼是 Next.js？

Next.js 是一個建立在 React 之上的全端框架，它提供了許多開箱即用的功能，包括：

- **伺服器端渲染 (SSR)**：提升首次載入速度和 SEO 優化
- **靜態網站生成 (SSG)**：預先生成頁面以獲得最佳性能
- **API 路由**：輕鬆建立後端 API 端點
- **檔案系統路由**：基於檔案結構自動生成路由
- **Edge Functions**：極速、全球分布式執行
- **Server/Client Components 混用**：最佳化效能與互動性

## Next.js 15 的新功能與範例

### 1. Partial Prerendering（PPR）

Next.js 15 最大亮點之一是 Partial Prerendering。你可以在同一頁面靜態渲染大部分內容，僅針對個別區塊（如登入狀態、推薦內容）動態渲染，實現極速首屏與動態互動兼得。

```tsx
// app/page.tsx
import { Suspense } from 'react';
import { UserPanel } from './UserPanel';

export default function Home() {
  return (
    <main>
      <h1>歡迎來到 Next.js 15！</h1>
      <Suspense fallback={<div>載入中...</div>}>
        <UserPanel /> {/* 僅這區塊動態渲染，其餘靜態 */}
      </Suspense>
    </main>
  )
}
```

### 2. Server Actions

Server Actions 讓你直接在伺服器端執行 mutation（如表單送出、資料寫入），無需額外 API Route，安全又簡單。

```tsx
// app/actions.ts
'use server';

export async function submitForm(data: FormData) {
  // 伺服器端處理資料
  await saveToDB(data);
}
```

```tsx
// app/page.tsx
import { submitForm } from './actions';

export default function ContactPage() {
  return (
    <form action={submitForm}>
      <input name="email" type="email" />
      <button type="submit">送出</button>
    </form>
  );
}
```

### 3. Turbopack 穩定版

Turbopack 取代 Webpack，正式進入穩定版。開發啟動、熱更新速度遠超以往，專案越大效益越明顯。

```bash
next dev --turbo
```

## 開始使用 Next.js 15

### 安裝

用最新版 create-next-app 快速建立專案：

```bash
npx create-next-app@latest my-app --typescript --tailwind --app
```

### 專案結構

```
my-app/
├── app/
│   ├── layout.tsx
│   ├── page.tsx
│   └── globals.css
├── public/
├── package.json
└── next.config.js
```

### 建立你的第一個頁面

在 `app/page.tsx`：

```tsx
export default function Home() {
  return (
    <main>
      <h1>歡迎使用 Next.js 15！</h1>
    </main>
  )
}
```

### 最佳實踐

1. **全面 TypeScript**：提升類型安全與 DX
2. **圖片優化**：用 Next.js Image 組件自動壓縮與 lazy load
3. **環境變數管理**：建議用 `.env.local`
4. **SEO 設定**：用新版 metadata API，支援 Open Graph/SEO/社群分享

```tsx
// app/layout.tsx
export const metadata = {
  title: '我的網站',
  description: '這是我的網站描述',
};
```
