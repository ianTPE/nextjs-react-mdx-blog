# 開始使用 Next.js 15

Next.js 15 正式登場，帶來更強大、更高效、更現代化的 React 全端開發體驗！本篇將帶你快速了解 Next.js 15 的最新特性、安裝方式與最佳實踐。

## 主要特性

- **Partial Prerendering（PPR）**
- **Turbopack 穩定版**
- **Server Actions**
- **Edge Runtime 強化**
- **MDX 2.0 支援**

## 安裝方法

```bash
npx create-next-app@latest my-app --typescript --tailwind --app
```

## 簡單範例

```tsx
// app/page.tsx
export default function Home() {
  return (
    <main>
      <h1>歡迎使用 Next.js 15！</h1>
    </main>
  )
}
```

## 結論

Next.js 15 以更快的開發體驗、更彈性的渲染模式與更強的全端能力，持續領先 React 生態圈。
